## 💼 REGULATORY FRAMEWORK COMPLIANCE

### Master Circular (Jan 30, 2026)
✅ Fully compliant with consolidation of all operative instructions through Dec 30, 2025
✅ Incorporates all 50+ rescinded earlier circulars
✅ Reflects Uniform Listing Agreement requirements

### Fifth Amendment (Nov 2025)
✅ Related Party Transaction thresholds aligned with Schedule XII (turnover-based)
✅ Enhanced Audit Committee approvals for subsidiary transactions
✅ Revised definitions of directors, KMP, relatives

### Industry Standards on Regulation 30 (Feb 2025)
✅ Four-tier disclosure timeline: 30 min / 12 hr / 24 hr / 2 days
✅ Materiality framework with quantitative & qualitative tests
✅ Defense for delayed disclosure documented
✅ "Credible and verifiable channels" SOP guidance

### Integrated Filing System (April 1, 2025)
✅ XBRL-based financial filing (PDF + XBRL mandatory)
✅ Governance disclosure in XBRL
✅ Eliminates separate utilities (all-in-one filing)
✅ API integration for high-volume filers

### BRSR Core Framework (FY 2023-24 onwards)
✅ 49 KPI structure (18 Environmental + 18 Social + 13 Governance)
✅ Phased applicability by market cap (top 1000 mandatory)
✅ 3rd-party assurance requirements (top 150 FY 2024-25+)
✅ Value chain ESG disclosure phased rollout

## 📊 USAGE SCENARIOS

### Scenario 1: Board Meeting Planning
**Timeline:** 7 days before meeting
1. Click "Create Event" → Select "Board Meeting"
2. Tool auto-calculates: Meeting date - 7 = notice date
3. Generates: Notice, Agenda, Disclosure Forms
4. CS downloads & sends to directors
5. Tool tracks: Notice sent, Disclosures received, Reg 30 needed?

### Scenario 2: Material Event Disclosure
**Timeline:** 30 mins / 12 hrs / 24 hrs (CRITICAL)
1. Board approves acquisition at 4 PM
2. CS selects "Material Event" in tool
3. Tool alerts: "30 MINUTES DEADLINE - Filing due by 4:30 PM"
4. CS customizes Regulation 30 template
5. Files with NSE/BSE within deadline
6. System records timestamp & compliance status

### Scenario 3: AGM Preparation
**Timeline:** 21 days notice
1. Create AGM event (June 1, 2026)
2. Tool generates: Notice, Proxy Form, Postal Ballot, Directors Report
3. CS reviews & customizes for company
4. Sends 21-day notice (by May 11)
5. Enables e-voting & postal ballot
6. After AGM, files Regulation 30 (12 hrs + 2 days for voting results)
7. Records all disclosures in system

### Scenario 4: BRSR Compliance
**Timeline:** Annual (FY ends March 31)
1. CS creates BRSR event (January 1, for data collection)
2. Tool displays 49 KPI framework
3. Data collection across departments (Q4 Jan-Mar)
4. Enter: Energy, emissions, diversity, safety metrics
5. Generate BRSR Core report (April)
6. Arrange 3rd-party assessment (May)
7. File with annual report (June 30)
